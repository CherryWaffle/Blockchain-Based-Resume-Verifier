# Backend (Node.js)

This is the Node.js backend for the Blockchain Resume Verifier. It uses Express, Web3, Pinata (IPFS) and MongoDB.

Quick start:

1. Copy `.env.example` to `.env` and fill values (MongoDB URI, Pinata keys, Alchemy URL, PRIVATE_KEY, CONTRACT_ADDRESS, CONTRACT_ABI_PATH)
2. cd backend
3. npm install
4. npm run dev
